Objetivos

O objetivo deste projeto foi de aplicar os conhecimentos adquiridos nas aulas de Fundamentos de Programação Web.

Público Alvo

  • O público alvo deste projeto foram todos aqueles que apreciam a história do futebol e que, de alguma maneira, buscam fazer parte dela.

Tecnologias Usadas

HTML 5

CSS

Java Script

Requisitos Mínimos

  • Dispositivos com largura de resolução de 330px
  • Chrome versão 22
  • Edge versão 12
  • Safari versão 7
  • Firefox versão 20
  • Opera versão 12.1
  • IE versão 10 (suporte parcial)

Dificuldades Encontradas

  • Criação dos formulários
  • Posicionamento dos elementos na página
